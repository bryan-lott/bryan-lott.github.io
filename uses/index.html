Tools I Use

Bryan Lott published on December 31, 2023
2 min, 267 words

Uses

A succinct overview of tools I use on a regular basis, mostly focused on my day job of software development but possibly other random stuff thrown in.

Software

  • Editor
    • NeoVim (AstroNvim)
    • VSCode: rarely used now
    • ~Emacs (Doom) https://github.com/doomemacs/doomemacs~
  • Browser
    • Brave
    • ~Firefox~
  • Utilities
    • DevUtils: One-off tools, base64 encode/decode, format and validate JSON, etc, really handy to have always running.
    • Amethyst: Automatically size/move windows
    • Raycast: Quick switcher and finder of files
    • GitKraken: The best (IMO) GUI git client
    • Syncthing: Dropbox kinda-sorta replacement
    • Blurred: Darken windows that aren't the active one (i.e., know where your cursor focus is)
    • Zola: Static site/blog generator
    • ~Finda: Find anything on your computer + quick switcher~
    • ~Quicksilver: Quick switcher~
  • Terminal
    • Alacritty w/ tmux or zellij
    • Wezterm
    • FishShell
    • Starship: Enhance your command line
    • ~zsh: Better than bash, more posix compliant than fish~
    • ~iTerm2: I'd like to use something else, but this works the best for now~

Services

  • ProtonMail: Personal email
  • YouNeedABudget: Finance tracking
  • Deezer: Music
  • Github: Off-computer code backup
  • Gitlab: Off-computer code backup, blog
  • Kagi: Search engine
  • Dropbox: Backup & sync
  • Bitwarden: Password manager
  • xBrowserSync: Sync your bookmarks across multiple devices and multiple browsers. Also offers a pretty killer search.
  • A Fine Start: pseudo-bookmarks on your new tab screen
  • ~1Password: If you're not using a password manager, you've already been compromised and just don't know it yet~
  • ~Feedly: I love my RSS feeds~

Hardware

  • MacBook Pro (i.e., whatever work provides me)
  • Alienware R7 (desktop/personal machine)
  • ZSA Voyager Keyboard
  • Streamdeck
  • Kensington Slimblade Trackball
  • Logitech MX Master 3
  • Remarkable2
  • Pixel 7 Pro
  • ~Ultimate Hacking Keyboard https://ultimatehackingkeyboard.com/~
